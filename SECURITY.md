# Security Policy

## Supported Versions

Dizznem Bot is a single-instance private Discord bot and does not follow a versioned release cycle.
Security fixes are applied directly to the main branch and deployed to the live bot.

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public GitHub issue.**
Public disclosure before a fix is in place puts the bot and its users at risk.

Instead, report it privately through one of the following:

- **Discord:** `karmasb`
- **Email:** mckinneyjaiden5418@gmail.com

### What to Include
- A clear description of the vulnerability
- Steps to reproduce it
- Any potential impact you've identified (e.g. token exposure, privilege escalation, data leak)

### What to Expect
- I'll acknowledge your report as soon as I can
- I'll keep you updated as I investigate and work on a fix
- Once resolved, I'm happy to credit you if you'd like

Thank you for helping keep Dizznem Bot secure.
